 Student Task Management Jira Project

 Overview

This project demonstrates the use of Jira for Agile Scrum project management. The project simulates a Student Task Management System and showcases how Features, Stories, Subtasks, Sprints, Dashboards, and Workflows are managed in Jira.


 Project Objectives
- Manage student accounts and authentication
- Create and manage tasks
- Track task status
- Monitor project progress using dashboards
- Apply Agile Scrum methodology


 Agile Methodology
- Scrum Framework
- Sprint Planning
- Backlog Management
- Story Tracking
- Dashboard Reporting


 Project Structure
 Feature 1: User Management
 Story: Student Registration
- Design Registration Form
- Validate User Inputs
- Create Registration Database Table
- Test Registration Process

 Story: Student Login
- User Authentication
- Secure Access Management


 Feature 2: Task Management

 Story: Create Task
- Design Task Creation Form
- Validate Task Inputs
- Save Task to Database
- Test Task Creation

 Story: Update Task Status

 Story: Delete Task
